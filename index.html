Introduction

A table can be used several ways on a page but one thing all tables have in common is that they are all used to present information in an organized way.


If you are looking to create a table on a web page, there are several things that you need to know.


One of the most important things you need to know before you can create a table using HTML are your table tags.

We are going to cover this later on as it is the most important thing you need to know when creating a table but before that, there are several things you need to be aware of before creating a table


  • You should know the element tags of a table
  • You should have a good idea of how you plan to arrange the table
What You Should Already Know

Building a table can be fun if you know what you are doing.


But before you jump in to start building a table there are several things that you should already know.


  • You should know how to use HTML to create content on a page
  • You should know how to use CSS as this would make it easy when customizing your table
HTML Table Tags

There are several tags that you need to know to create a table.


I would list and explain some of these tags below and later leave a reference where you can get more information on creating a table.

  • <table></table>

    • This is considered the first tag to use when creating a table. Nothing happens here until you place other tags within it. All this tag does is that it creates a table where you are to place your content

  • <tr></tr>

    • This adds tags for rows. Depending on how many you want would determine how many of its opening and closing tags you would include.

    Just using this tag would make any data display on the table. You must define each cell element and you would add data using the table data element <td>

  • <td></td>

    • As we already mentioned, this allows us to input data with the tags which would then appear on the form of cells. Below is a representation of a simple table with styling.

  • <th></th>

    • We add headings to the contents of a table by making use of this tag. It must be placed within the <tr></tr> tag
HTML Table Scope Attribute

Two scope attributes that you can use in your table are:

  • row: by using this value, it makes it clear that the heading is for a row
  • col: this value makes it clear that the heading is for a column

    • There are other attributes that we use when creating a table

  • colspan: we use this attribute to create a column to span more than one cell. Using this attribute creates data that spans multiple columns and it accepts an integer as a value to denote the number of columns it would occupy
  • rowspan: This works in a similar way to colspan but instead of spanning through columns it spans through row.

Semantic Tags

Semantic elements clearly describes its meaning to both the browser and the developer. They provide information about the content between the opening and closing tags.


TAGS NAME DESCRIPTION
<header> Header tag This is usually for navigational links or introductory content containing h1 to h6 headings
<main> Main tag It is used to encapsulate the dominant content within a webpage. This element wraps the main content of the page, and there should be only one per page.
<article> Article tag It used to to wrap independent, self-contained content. This tag works well with blog entries, forum posts, or news articles.
<section> Section tag Groups related content. It is used for grouping thematically related content.
<nav> Navigation tag This is used to define a block of navigation links such as menus and tables of contents. It can be used inside the header element and also on its own.
<footer> Footer tag Mostly located at the bottom of the page outside the <main> tag and is used to hold info such as contact information, copyright information, Terms of use, Site Map, Reference to top page links, etc.
<aside> Aside tag This is used to mark additional information that can enhance another element but isn't required in order to understand the main content.
<Figure> Figure tag It is an illustration used to encapsulate media such as image, illustration, diagram, code snippet, etc.
<Figcaption> Figcaption tag This is used to add a caption to the image as it is an element used to describe the media in the <figure tag>.

Non-Semantic Tags

These tags tell us nothing about their contents.


TAGS NAME DESCRIPTION
<div> Division tag This defines a division or a section in an HTML document. It is used as a container for HTML elements
<span> Span tag This is an inline container used to mark up a part of a text, or a part of a document
